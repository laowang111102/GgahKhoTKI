# 项目概述 python1120

本项目是一个基于Python和Django框架的Web应用程序，旨在实现一个书籍信息管理系统。该系统遵循MVC架构模式，实现了前后端分离，具有清晰的功能模块划分。

## 技术栈

- **架构**: B/S、MVC
- **开发语言**: Python
- **开发框架**: Django
- **数据库**: MySQL

## 系统角色

系统定义了两个主要角色：用户和管理员。

### 用户

- 查看书籍信息
- 搜索和分类过滤书籍
- 管理个人信息
- 查看和管理购物车
- 查看订单信息

### 管理员

- 查看系统主页
- 管理书籍信息
- 查看和管理订单
- 处理退订请求
- 管理用户信息

## 功能模块

### 前台系统

- **首页**: 展示所有书籍信息，支持搜索和分类过滤。
- **用户中心**: 用户可以查看和管理个人信息。
- **我的购物车**: 用户可以查看购物车中的书籍，进行结算。
- **我的订单**: 用户可以查看已提交的订单信息。
- **商品分类**: 用户可以通过分类查看书籍。

### 后台管理系统

- **首页**: 管理员登录后可查看系统主页面。
- **书籍管理**: 管理员可以查看和管理所有书籍信息，支持添加、修改和删除书籍。
- **订单管理**: 管理员可以查看所有订单信息，支持订单状态的更新。
- **退订管理**: 管理员可以处理用户的退订请求。
- **用户管理**: 管理员可以查看和管理用户信息。

## 运行环境

- **操作系统**: Windows
- **开发工具**: PyCharm
- **Python版本**: 3.x
- **MySQL版本**: 5.x
